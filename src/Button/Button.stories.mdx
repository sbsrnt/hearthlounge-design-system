# Button

## Defines a clickable button.

<Preview withToolbar>
  <Story name="default">
    <Button>Click Me</Button>
  </Story>
</Preview>

<Props of={Button} />

## Usage

Inside a `Button` component you can put content, like text or images and fire actions on click.

### Sizes

By default the size of the button is `medium`.
There are `2` supported sizes.

<Story name="sizes">
  <div>
    <Button className="mr-2">Click Me</Button>
    <Button size="small">Click Me</Button>
  </div>
</Story>

### Themes

By default the theme of the button is `primary`. There are `2` themes are supported.

<Story name="themes">
  <div>
    <Button className="mr-2">Click Me</Button>
    <Button theme="secondary">Click Me</Button>
  </div>
</Story>

### Loading

The loading state is used when action is async and the
response was not yet received, hence the loading usage.

<Story name="loading">
  <Button loading>Click Me</Button>
</Story>
