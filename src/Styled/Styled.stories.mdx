# Styled

## Passes dynamic styles to all child components

By default `<Styled />` has styles as the component would be inactive.

<Preview withToolbar>
  <Story name="default">
    <Styled withHover>
      <Text>Hover over me</Text>
    </Styled>
  </Story>
</Preview>

<Props of={Styled} />

### `withTransition`

<Preview withToolbar>
  <Story name="withTransition">
    <Styled withHover withTransition>
      <Text>Hover over me</Text>
    </Styled>
  </Story>
</Preview>
