# HearthLounge's Color Palette

Color names are based on [Blizzard's Hearthstone official API `slug` names](https://develop.battle.net/documentation/api-reference/hearthstone-game-data-api).

The default color for each category is `{color}50`.

Not supported sets from `Year of the Raven` and `Year of the Dragon`. Yet.
